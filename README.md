# Vulkan Triangle

该项目是使用纯 `Vulkan` 接口在 `Windows` 和 `Linux` 平台下渲染一个三角形的示例程序。只用于学习，禁止用于商业。

## 说明

不需要配置任何环境，直接使用 `CMake` 导出工程编译即可。

将会输出 `VulkanTriangle.exe` 可执行程序。
